# Sauda AI — маркетинговый лендинг

Лендинг сервиса **Sauda AI**: «Фотографируешь изделие, отвечаешь на 3 вопроса —
получаешь готовую карточку товара для маркетплейса за 10 секунд».
Сделан для мастеров и фермеров Казахстана.

> Это только фронтенд-макет: без бэкенда, базы данных и реального ИИ.
> Форма регистрации проверяет данные на стороне браузера и показывает экран успеха.

## Технологии

- **React + Vite + TypeScript** — каркас приложения
- **Tailwind CSS v4** — стили (подключён через плагин `@tailwindcss/vite`)
- **lucide-react** — иконки
- **framer-motion** — плавные анимации появления секций
- Шрифты **Manrope** (заголовки) и **Inter** (текст) через Google Fonts
- Мультиязычность RU/KZ — простой объект-словарь, без i18n-библиотек

## Как запустить

```bash
# 1. Установить зависимости (один раз)
npm install

# 2. Запустить сайт для разработки
npm run dev
```

После запуска откройте в браузере адрес, который покажет Vite
(обычно http://localhost:5173).

Сборка для публикации: `npm run build` (результат появится в папке `dist/`).

## Структура проекта

```
src/
  content.ts              — ВСЕ тексты сайта на русском и казахском
  App.tsx                 — выбор языка + переключение экранов (лендинг/регистрация)
  index.css               — подключение Tailwind, фирменные цвета и шрифты
  components/
    Navbar.tsx            — липкая шапка: меню, переключатель KZ/RU, кнопка CTA
    Hero.tsx              — главный экран с мокапом карточки товара
    ProblemSection.tsx    — «Знакомо?»: боли аудитории
    HowItWorks.tsx        — 3 шага работы сервиса
    Features.tsx          — 4 возможности + бейдж «на казахском и русском»
    BeforeAfter.tsx       — сравнение карточки «до» и «после»
    Audience.tsx          — для кого сервис: 3 персоны
    FAQ.tsx               — аккордеон с частыми вопросами
    CTASection.tsx        — финальный призыв к действию
    Footer.tsx            — подвал с ссылками-заглушками
    RegisterPage.tsx      — экран регистрации с клиентской валидацией
    FadeIn.tsx            — маленькая обёртка для анимации появления
```
